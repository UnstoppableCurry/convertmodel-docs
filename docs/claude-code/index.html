Path 01

Claude Code 直连

最短可用路径。先完成安装和最小配置，再做模型切换与排错。

安装

  1. 安装 Node.js 18+。
  2. 执行 npm install -g @anthropic-ai/claude-code。
  3. 初始化后确认 ~/.claude 配置目录已创建。

最小配置

{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "YOUR_API_KEY",
    "ANTHROPIC_BASE_URL": "https://convertmodel.net/anthropic",
    "ANTHROPIC_MODEL": "claude-sonnet-4-6"
  }
}

路径速查

macOS / Linux

~/.claude/settings.json

Windows

%USERPROFILE%/.claude/settings.json

使用建议

  • 默认模型建议使用 claude-sonnet-4-6。
  • 高强度任务再切到 Opus 档位。
  • 如果出现 503，先检查分组 key 与模型名是否匹配。
