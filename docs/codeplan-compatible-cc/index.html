CodePlan 兼容 Claude Code

这是 Claude Code 客户端 + Anthropic 兼容接口 + 网关模型映射的组合路径。客户端配置方式与直连一致，差异在上游映射与调度。

配置示例

{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "YOUR_API_KEY",
    "ANTHROPIC_BASE_URL": "https://convertmodel.net/anthropic",
    "ANTHROPIC_MODEL": "claude-sonnet-4-6"
  }
}

当前映射

Sonnet

claude-sonnet-4-6 -> kimi-k2.5

Opus

claude-opus-4-6 -> glm-5

关键注意事项

  • 优先使用 Anthropic 分组 key，不要混用 OpenAI 分组 key。
  • 请求路径是 /v1/messages。
  • 若调度前失败，使用记录可能没有落库，需要查看网关日志。
  • 503 No available accounts 常见于模型名、分组或账号过滤不匹配。

排错顺序：先看请求模型名，再看上游映射，再看账号过滤与可用容量。
