Codex 兼容 CC 网关

保留 Codex CLI 使用习惯，只替换到统一网关。文档示例采用中立官方 CLI 配置，不使用品牌定制命令。

安装

  1. 安装 Node.js 18+。
  2. 执行 npm install -g @openai/codex@latest。
  3. 写入 auth.json 与 config.toml。

auth.json

{
  "OPENAI_API_KEY": "YOUR_API_KEY"
}

config.toml

model_provider = "convertmodel"
model = "gpt-5.4"
model_reasoning_effort = "low"

[model_providers.convertmodel]
name = "convertmodel"
base_url = "https://convertmodel.net"
wire_api = "responses"
requires_openai_auth = true

适用边界

  • Codex 与 Claude Code 是不同客户端链路，配置文件也不同。
  • 如果目标是稳定 Claude 风格编码体验，优先使用 Claude Code 直连。
  • 如果你已有 Codex 工作流，可用本页方式接入同一网关。
