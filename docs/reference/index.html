路径、变量与排错速查

用于快速复制与比对。出现故障时，先走最小排查路径再深入日志。

配置路径

Claude Code

  • macOS/Linux: ~/.claude/settings.json
  • Windows: %USERPROFILE%/.claude/settings.json
  • onboarding: ~/.claude.json

Codex

  • auth.json
  • config.toml
  • 建议与项目凭据分离管理。

常见错误

  • 503 No available accounts：先查模型名、分组 key、账号过滤。
  • 配置无效：先查文件路径、变量名拼写、优先级覆盖。
  • 使用记录无条目：请求可能在调度前失败，需看网关日志。
  • 模型行为不符预期：先看客户端请求模型名，再核对映射。
最小排查路径：1) 请求模型名 2) 分组与 key 3) 上游映射 4) 账号可用容量 5) 网关日志。
